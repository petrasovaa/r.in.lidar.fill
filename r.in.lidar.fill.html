DESCRIPTION

r.in.lidar/fill calls r.in.lidar and then fills the NULL values using r.fill.gaps addon. The filling process has 2 phases, in the first only the small gaps are filled, and in the second bigger holes are filled based on specified parameters distance and power.

NOTES

Filling bigger gaps does not always produce intended result, changing distance and power can help.
